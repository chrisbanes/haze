VisualEffectContext

interface VisualEffectContext

Context provided to VisualEffect implementations during their lifecycle.

This interface abstracts away the underlying node implementation, providing effects with all the geometry, configuration, and accessor capabilities they need without direct coupling to HazeEffectNode.
