invalidateOnHazeAreaPreDraw

actual fun invalidateOnHazeAreaPreDraw(): Boolean

We need to manually invalidate if the HazeSourceNode 'draws' on certain API levels:

  • API 31: Ideally this wouldn't be necessary, but its been seen that API 31 has a few issues with RenderNodes not automatically re-painting. We workaround it by manually invalidating.

  • Anything below API 31 does not have RenderEffect so we need to force invalidations.
