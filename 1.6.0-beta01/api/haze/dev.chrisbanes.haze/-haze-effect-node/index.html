HazeEffectNode

class HazeEffectNode(var state: HazeState, style: HazeStyle = HazeStyle.Unspecified, var block: HazeEffectScope.() -> Unit? = null) : Modifier.Node, CompositionLocalConsumerModifierNode, GlobalPositionAwareModifierNode, LayoutAwareModifierNode, ObserverModifierNode, DrawModifierNode, TraversableNode, HazeEffectScope

The Modifier.Node implementation used by Modifier.hazeEffect.

This is public API in order to aid custom extensible modifiers, but we reserve the right to be able to change the API in the future, hence why it is marked as experimental forever.
