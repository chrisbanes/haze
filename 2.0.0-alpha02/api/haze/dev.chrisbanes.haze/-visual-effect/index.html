VisualEffect

interface VisualEffect

A visual effect that can be applied to content behind or in front of a composable.

Implementations receive a VisualEffectContext during their lifecycle which provides access to geometry, configuration, and platform capabilities without direct coupling to the underlying node implementation.

VisualEffect instances are single-owner and must not be attached to multiple Modifier.hazeEffect nodes at the same time. Reusing the same effect instance across concurrently active nodes will throw an IllegalStateException.
