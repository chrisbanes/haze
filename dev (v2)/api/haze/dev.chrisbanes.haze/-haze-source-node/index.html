HazeSourceNode

class HazeSourceNode(state: HazeState, zIndex: Float = 0.0f, key: Any? = null) : Modifier.Node, CompositionLocalConsumerModifierNode, GlobalPositionAwareModifierNode, LayoutAwareModifierNode, DrawModifierNode, TraversableNode, ObserverModifierNode

The Modifier.Node implementation used by Modifier.hazeSource.

This is public API in order to aid custom extensible modifiers, but we reserve the right to be able to change the API in the future, hence why it is marked as experimental forever.
